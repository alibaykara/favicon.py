# favicon.py

favicon.py Favicon Hash Scanner


## Run Locally

Clone the project

```bash
  git clone https://github.com/alibaykara/favicon.py.git
```

Go to the project directory

```bash
  cd favicon.py
```

Install dependencies

```bash
  pip install mmh3
```

Start the python3

```bash
  python3 favicon.py https://www.google.com/favicon.ico
```
